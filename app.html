How to use

  1. Add a two-sided or one-sided document card using the buttons below.
  2. Upload front (and back for two-sided). PDFs are auto-converted to images.
  3. Enable the signature checkbox to add a signature image. Use the 🎯 Place Signature button to position it precisely using arrow keys (hold Shift for bigger steps).
  4. Background removal note: the default "Local AI" remover runs entirely on your device — nothing is uploaded. There's also an optional remove.bg API mode (your own key) for tougher photos; choosing it uploads that photo to remove.bg's external servers. It's off by default and we'll ask you to confirm before the first upload.
  5. Click Download and choose PDF, JPG, or PNG output format.
  6. Use Single-Doc Tools (bottom right) to crop, rotate, flip, compress and convert any image or PDF all at once before sending it here.
